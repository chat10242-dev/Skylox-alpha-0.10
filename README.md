# 🌌 SKYLOX — Alpha 0.10
La piattaforma neon che unisce giochi, multiplayer P2P, pseudo‑multiplayer, editor ruoli, engine importer e un sistema di account multi‑utente.  
Questa è la **versione Alpha 0.10**, una build dimostrativa delle fondamenta della futura console virtuale **SkyloxOS**.

---

## 🚀 Funzionalità principali
- **Login + multi‑account**
- **Importazione giochi .exe**
- **Importazione engine .exe**
- **Multiplayer P2P (Host / Join)**
- **Share Play con input remoto**
- **Pseudo‑multiplayer avanzato**
  - Editor ruoli
  - Editor input per Player 2
  - Editor abilità
  - Modalità spettatore
- **Chat globale**
- **Feed attività**
- **Temi, lingua, controller, prestazioni**
- **Sistema JSON persistente**
- **UI neon stile console**

---

## 🧩 Pseudo‑Multiplayer
Skylox permette di giocare in coppia anche nei giochi **single‑player**, grazie a:

- Share Play P2P
- Input remoto
- Ruoli configurabili
- Abilità separate
- Modalità spettatore

Il pseudo‑multiplayer **non modifica i giochi**: li rende cooperativi tramite input condivisi.

---

## 🎮 Multiplayer P2P
Skylox include un sistema multiplayer semplice e diretto:

- Host → crea la stanza
- Join → si collega tramite IP
- Chat Share Play
- Log connessione
- Input remoto

Perfetto per testare giochi, prototipi e modalità coop.
